This folder contains all current documentation on files developed for the project.
It has been split into four distinct categories: File Organization, Tutorials, VHDL Code, and ZPUino Code.

The File Organization file details how all of the files are organized and where to look for information, if available.
The Tutorials file contains tutorials written for the project intended to get new team members started working on the Papilio.
The VHDL Code file outlines what VHDL code is running on the robot and what it is doing.
THe ZPUino Code file outlines what is being run on the ZPUino soft processor and how it is organized.
